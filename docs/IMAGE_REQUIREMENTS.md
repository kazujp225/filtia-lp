# Filtia LP 画像生成依頼書

## 1. プロジェクト概要

**サービス名**: Filtia（フィルティア）
**サービス内容**: 法人向け浄水型ウォーターサーバー
**LPの目的**: 営業商談後に閲覧され、稟議を通すための「信用と理解を揃える装置」

---

## 2. デザインコンセプト

> **Quiet Premium Water**
> 清潔・静けさ・秩序。水色で"水っぽさ"を押し付けず、余白と質感で"水の透明感"を出す。

### トーン＆マナー

| 方向性 | 表現方法 |
|--------|----------|
| 清潔感 | 白ベース |
| 高級感 | 黒/チャコール/グレージュをアクセント |
| 水の表現 | 水色の多用は避ける。余白、透明感、薄いグラデ、写真の光で表現（ベタな青で水を表現しない） |

### カラーパレット

| 役割 | カラー名 | HEX |
|------|----------|-----|
| Base（背景） | オフホワイト | `#F7F8FA` |
| Surface（カード） | ホワイト | `#FFFFFF` |
| Text（主文） | チャコール | `#1F2937` |
| Text（補助） | スレート | `#475569` |
| Accent（ブランドカラー） | くすみアクア | `#4BA3B6` |
| Accent Dark | ディープティール | `#0F4C5C` |
| Trust（安全・強調） | ネイビー寄り | `#0B2A3C` |

### 写真ルール

| 項目 | ルール |
|------|--------|
| 光 | 自然光っぽい柔らかさ（影は硬くしない） |
| 背景 | 白〜グレージュ、木目は"薄く" |
| 画角 | 正面/45度/クローズアップを揃える |
| レタッチ | 彩度低め、コントラスト控えめ（ギラつき禁止） |

### NG事項

- 水しぶきCGは使わない（B2C臭が出る）
- ベタな水色グラデ全面・青の面積が広すぎる
- 原色ブルーのアクセント
- 派手なキャンペーン風の装飾

## 3. 必要な画像一覧

### A. 商品カット【最優先】

#### A-1. ヒーロー用メイン商品画像
| 項目 | 仕様 |
|------|------|
| タイプ | ロングタイプ（床置き） |
| カラー | ホワイト、ブラック 各1枚 |
| 画角 | 正面または45度 |
| 背景 | 完全白 or 薄グレー |
| 用途 | ファーストビュー（FV）セクション |
| サイズ | 横2400px以上推奨 |
| 形式 | WebP + 透過PNG |

#### A-2. ディテールカット（説得力向上用）
| 部位 | 説明 |
|------|------|
| 注水口まわり | 衛生的な印象を訴求 |
| 操作パネル | 温水/冷水のUI、温度表示 |
| タンク・内部 | 見せられる範囲で |
| カートリッジ | 単体 + 装着イメージ |

#### A-3. サイズ感カット
- 人の手との比較
- または椅子/デスク横に置いた写真
- ※過剰な生活感は出さない

---

### B. 設置イメージ【B2B訴求に重要】

#### B-1. オフィス導入シーン（3パターン）

| シーン | 説明 | ポイント |
|--------|------|----------|
| 受付 | 来客対応イメージ | 高級感、おもてなし感 |
| 会議室 | 打ち合わせ中 | ビジネスシーン、知的な雰囲気 |
| 休憩スペース | 従業員利用 | リラックス感、福利厚生感 |

**構図の注意点**:
- サーバーが主役だが、空間の品位を崩さない
- 人物は入れても入れなくてもOK（入れる場合はビジネスカジュアル）
- オフィス家具は上質なものを想定

#### B-2. メンテ・運用シーン
| シーン | 説明 |
|--------|------|
| カートリッジ交換 | 手元のみ（顔不要）、簡単さを訴求 |
| 日常の拭き取り | 手軽なお手入れ感 |

---

### C. 図解・アイコン素材

#### C-1. 浄水の仕組み図（シンプル）
```
水道水 → フィルター → タンク → UV殺菌 → 浄水
```
- テイスト: 線画 + 最小限のアクセントカラー（`#4BA3B6`）
- モダン・クリーンな印象

#### C-2. 導入ステップアイコン（4つ）
| Step | 内容 | アイコンイメージ |
|------|------|-----------------|
| 1 | お問い合わせ | 吹き出し/メール |
| 2 | ヒアリング・ご提案 | クリップボード/書類 |
| 3 | お届け・設置 | 配送トラック/箱 |
| 4 | 運用開始 | チェックマーク/水滴 |

#### C-3. 特徴アイコン（ピクトグラム）
| 特徴 | アイコンイメージ |
|------|-----------------|
| 工事不要 | ×工具、コンセントのみ |
| 温冷対応 | 温度計/水滴（赤青） |
| 静音 | 音符×、静けさ表現 |
| 節電 | 葉っぱ/エコマーク |
| 除去 | フィルター/盾 |

**アイコンスタイル**:
- 線画（ストローク1.5px程度）
- カラー: `#4BA3B6` または `#0F4C5C`
- 統一感のあるデザイン

---

### D. 信頼素材（あれば）

| 素材 | 説明 |
|------|------|
| 導入企業ロゴ | 許諾取得済みのもののみ |
| 認証マーク | JIS規格等（あれば） |
| 試験結果 | 要点を図表化（PDFそのままは避ける） |

## 4. 画像仕様まとめ

| 項目 | 仕様 |
|------|------|
| 形式 | 写真: WebP（品質75〜85）、透過: PNG、ロゴ/アイコン: SVG |
| 解像度 | 横1600px以上（ヒーローは横2400px推奨） |
| アスペクト比 | 16:9（背景）、4:3（カード/シーン）、1:1（サムネ） |
| Retina対応 | 重要カットは2x版も作成 |

## 6. 優先度

1. **最優先**: メイン商品画像（FV用）
2. **高**: 設置シーン写真（受付・会議室・休憩室）
